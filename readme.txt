텐서플로우 함수

tf.add	덧셈을 수행합니다.
tf.subtract	뺄셈을 수행합니다.
tf.multiply	곱셈을 수행합니다.
tf.truediv	나눗셈의 몫을 구합니다.
tf.mod	나눗셈의 나머지를 구합니다.
tf.abs 절대값을 구합니다.

tf.negative	음수를 반환합니다.
tf.sign	부호를 반환합니다. (음수: -1, 양수: 1, 0: 0)
tf.square	제곱을 수행합니다.
tf.sqrt	제곱근을 반환합니다.
tf.pow	거듭제곱을 수행합니다.
tf.maximum 더 큰 값을 반환합니다.
tf.minimum	더 작은 값을 반환합니다.
tf.exp	지수 값을 계산합니다.
tf.log	로그 값을 계산합니다.
tf.sin	사인 함수 값을 계산합니다.
tf.cos	코사인 함수 값을 계산합니다.
